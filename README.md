# Presonalize_analizer/ Mr analyst


## Overview

Mr. Analyst is an AI-powered document analysis application developed using the Retrieval-Augmented Generation (RAG) approach. The application enables users to upload PDF and CSV files, ask questions in natural language, and receive accurate answers based on the uploaded documents. Instead of manually searching through large files, users can quickly retrieve relevant information with the help of Artificial Intelligence.

---

## Why This Project?

Finding information in lengthy documents or datasets is often time-consuming and inefficient. This project was created to simplify document analysis by allowing users to interact with their documents through simple questions.

The main objectives of this project are:

* Reduce the time required to search through documents.
* Make document analysis simple and interactive.
* Demonstrate the practical implementation of Retrieval-Augmented Generation (RAG).
* Learn and apply modern Generative AI technologies.
* Build a user-friendly AI assistant for document-based question answering.

---

## What Can Mr. Analyst Do?

* Upload multiple PDF documents.
* Upload CSV datasets.
* Extract text from uploaded documents.
* Split documents into meaningful text chunks.
* Generate embeddings for semantic search.
* Store embeddings in a ChromaDB vector database.
* Retrieve the most relevant document content.
* Generate AI-powered answers using a Large Language Model.
* Display the retrieved context for better transparency.
* Reset the knowledge base whenever needed.

---

## Results Achieved So Far

The current version of the project successfully provides the following functionalities:

* Successfully processes PDF and CSV files.
* Creates vector embeddings for uploaded documents.
* Stores document embeddings in ChromaDB.
* Retrieves relevant document chunks based on user queries.
* Generates context-aware answers using the Groq Llama model.
* Supports multiple document uploads.
* Provides an interactive Streamlit-based user interface.
* Implements the complete Retrieval-Augmented Generation (RAG) workflow.
